Asteroids


The first game that I made completely by myself - it was programmed in java using the Light weight java game library. I wrote the engine myself - which included writing the rendering code, and the physics.I also created the art assets myself(you can probably tell) and wrote all the game logic as well. Sounds where sourced from creative commons.

The object of the game is simple - shoot the asteroids to avoid colliding with them.
